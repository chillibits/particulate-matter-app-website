FAQ

How can I install the Particulate Matter App on my device?

Navigate your Android device to the Google PlayStore and search for 'Particulate Matter App'. Tap on the first search result and then on 'Install'. The app gets now downloaded and installed on your device. Tap on 'Open', to launch the app.

Info: Currently the app only support Android devices. If you don't have an Android device, you can try our web app which has the same features.

How can I link a pm sensor?

There are two different ways to link an pm sensor within the app:

Add sensor to the map
Navigate to the 'All sensors' tab and search for a sensor, which you want to add to your favourites. Tap on the marker and on 'Add favourite'. Finally, you can choose an custom name for the sensor. As soon as the sensor is linked, it can be seen on the tab 'My favourites'.

Link own sensor
Navigate to the tab 'My sensors' and tap on the '+' button at the bottom right. Fill in the data of the sensor by truth. We ask you, to specify the precise location of it. Please tap on the tiny done icon in the upper right corner. Now, the sensor should be found on the tab 'My sensors'.

How do I edit an existing sensor?

Change to the tab, where your sensor gets displayed and tap on the three dots to the right of the sensor item (For own sensors, you have to navigate to the 'My sensors' page). Tap on 'Edit sensor'. As soon as you finished to update the sensor, tap on the done icon in the upper right corner.

How do I remove an existing link of a sensor?

Change to the tab, where your sensor gets displayed and tap on the three dots to the right of the sensor item (For own sensors, you have to navigate to the 'My sensors' page). Tap on 'Remove sensor'. Confirm the dialog by tapping 'Remove sensor'.

How can I show the measurement data of a specific sensor?

Here, there are two ways. Either you create a link (see above) and tap on it or navigate to the map, tap on a sensor and select 'Show measurements'.

How do I enable the 'tick' in the sensor configuration?

To enable the 'tick' in the sensor configuration, you have to be in the same local network with the sensor (e.g. same local WiFi network). Please enter the IP address of the sensor in the address bar of your browser. There should be several buttons visible. Tap on 'Configuration' and navigate to the tab 'APIs'. Enable the option 'Send to Feinstaub-App' and tap 'Save and restart'. It can last up to 10 minutes to receive any data from the sensor. Please be patient until you can link the sensor within the app.
